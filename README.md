# 🍕 Pen2Pizza

A full-stack college project website built using **HTML, CSS, JavaScript, Node.js, Express.js**.

This project includes features like:

- 🔐 OTP based signup 
- 👤 Login system
- 🛍 Product listing
- 🛒 Add to cart
- ⭐ Reviews
- 💬 Chat option
- 💳 Payment integration UI
- 🚀 Live deployment ready

---

## 📌 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

---

## 🔑 Features

### User Authentication
- Signup with OTP verification
- Login with email and password

### Product Features
- Add product
- View products
- Search products
- Add to cart
- Write reviews

### Extra Features
- Chat page
- Payment page UI
- Seller / freelancer pages
